# WEB_Tourism

Репозиторий дря работы над web-приложением для школы туризма. 
Структура приложения: https://boards.yandex.ru/whiteboard/?hash=9c9ce9bc95984e82d0040ca347134d21

Для правильной сборки:
1. Копируем ссылку на репозиторий https://github.com/podbor16/WEB_Tourism.git
2. Клонируем репозиторий в папку на компе `git clone https://github.com/podbor16/WEB_Tourism.git`
3. Активируем виртуальное окружение
4. Устанавливаем requirements проекта `pip install -r requirements.txt`
5. Для запуска сервера react перейти в папку web_tourism/frontend и запустить команду `npm start`
6. Для запуска сервера django перейти в корневую папку проекта (web_tourism, в ней файл manage.py лежит) и запустить команду `python manage.py runserver`
